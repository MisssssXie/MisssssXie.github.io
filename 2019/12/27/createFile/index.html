Hexo如何在終端機建立md類型文章

發表於 2019-12-27 | 分類於 Hexo

開始實作

1. 到終端機下進入自己的Hexo站點目錄，並建立新檔案

$ cd your-hexo-site // blog
$ hexo new createFile // createFile是文章檔案名稱

2. 到Hexo站點目錄下，進入Sourcea資料夾，找到「_posts」資料夾並點開createFile.md檔修改內容

3. 回到終端機下部署剛剛建立好的資料夾囉

$ sudo hexo d -g

4. 最後啟動伺服器，預設是 http://localhost:4000 立即觀看改變效果

$ sudo hexo s
語法 註解
hexo d ｄeploy 部署
hexo g generate 部署網站前先產生靜態檔案
hexo s server 啟動伺服器

Hexo 建立「子分類」資料夾

發表於 2019-03-28 | 分類於 Hexo

前言碎碎念

嗨，大家好！初次接觸Hexo寫自己的Blog，就先寫寫一些環境建置小技巧熟悉一下囉～本篇就來記錄如何在「categories」大分類下建立子分類，大家也可以參考看看：]

成果

流程概述

  1. 建立「TensorFlow」資料夾
  2. 更改資料夾內「index」md 檔敘述
  3. 部署

開始實作

1. 到終端機下進入自己的Hexo站點目錄，並建立新頁面，名為「TensorFlow」

$ cd your-hexo-site
$ hexo new page TensorFlow

2. 到Hexo站點目錄下，進入Sourcea資料夾，找到剛剛建立的「TensorFlow」並點開index.md檔修改內容

---
title: TensorFlow
date: 2019-03-30 00:34:34
type: "TensorFlow"
comments: false
---

注意:

  1. type敘述是主資料夾，若加雙引號就是標記要在主資料夾(categories)下建立子資料夾(TensorFlow)。
  2. 頁面也會帶有評論。若需要關閉的話，請加入字段comments並將值設置為false。

3. 回到終端機下部署剛剛建立好的資料夾囉

$ sudo hexo d -g

4. 最後啟動伺服器，預設是 http://localhost:4000 立即觀看改變效果

$ sudo hexo s
語法 註解
hexo d ｄeploy 部署
hexo g generate 部署網站前先產生靜態檔案
hexo s server 啟動伺服器
